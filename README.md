# Supply Chain Performance Analysis (Interactive Dashboard using  PowerBI)
## Project Objective
The goal of this project is to analyze the end-to-end supply chain operations to identify bottlenecks in order fulfillment, optimize inventory levels, and reduce shipping delays to improve overall operational efficiency.
## Data Set Used
[Click here to access the Supply Chain Dataset](https://github.com/prathamthakar/Supply_chain_dashboard/commit/4456e7e8eeab758e22e81aa102e15916af151fd6)
## Key Performance Indicators (KPIs)
Order Fulfillment Rate: What percentage of orders are successfully delivered vs. cancelled or returned?

Lead Time Analysis: Which shipping carriers or routes have the highest delays?

Inventory Turnover: Which product categories are moving fastest (high demand) vs. slow-moving stock?

Shipping Cost by Carrier: Which logistics partner is most cost-effective?

Regional Distribution: Which states/regions face the most supply chain disruptions?

Stock-out Risk: Identification of products with low inventory levels relative to demand.
## Process
Data Cleaning: Handled inconsistent formatting in "Shipping Dates" and "Product IDs" to ensure accurate lead-time calculations.

Data Processing: Created calculated columns for "Days to Ship" and "Order Accuracy" using Excel formulas.

Data Analysis: Utilized Pivot Tables to categorize inventory into High, Medium, and Low velocity groups.

Data Visualization: Developed a dynamic dashboard featuring Slicers for Region, Product Category, and Shipping Mode.

## Project Insights
Delivery Bottlenecks: Standard shipping accounts for 70% of delays; upgrading top-tier customers to Express could improve satisfaction by 15%.

Inventory Imbalance: 30% of warehouse space is occupied by "Slow-Moving" electronics, while "Fast-Moving" apparel often faces stock-outs.

Geographic Trends: The North region has the highest return rate due to a specific logistics partner's handling issues.
## Final Conclusion
To optimize the supply chain, the business should renegotiate contracts with underperforming carriers in the North region and implement a Just-In-Time (JIT) inventory model for the electronics category to free up capital and warehouse space.
